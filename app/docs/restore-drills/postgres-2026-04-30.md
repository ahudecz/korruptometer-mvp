# Postgres restore drill — 2026-04-30

Launch-gate T126. Verifies that we can restore a Supabase Postgres snapshot
to a fresh branch and run smoke tests against it within an acceptable
window (FR-056, SC-020).

## Status

`PENDING` — must be exercised on staging before spec-Phase-2 ships.

## Procedure

1. Take a snapshot of the production Postgres via the Supabase dashboard
   (Database → Backups → "Take a backup").
2. Restore the snapshot into a new Supabase branch:
   `supabase branches create restore-drill-2026-04-30 --from-snapshot <id>`.
3. Wait for the branch to be ready. Record `time-to-restore`.
4. Run the smoke suite against the branch URL:
   * `pnpm --filter @korr/web test` (vitest, no DB)
   * `pnpm --filter @korr/db db:check` (schema diff)
   * Submit one test report through `/bejelentes` against the branch
     URL; assert it reaches `/admin` for the seeded admin.
5. Tear down the branch: `supabase branches delete restore-drill-2026-04-30`.
